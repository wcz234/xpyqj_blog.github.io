## 📝 使用建议

### 日常开发

1. **使用 Makefile 命令**
   ```bash
   make new TITLE="文章标题"    # 创建文章
   make dev                     # 开发预览
   make all                     # 一键部署
   ```

2. **图片管理规范**
   - 封面图放在 `source/images/covers/`
   - 文章配图放在 `source/images/posts/文章名/`
   - 图标放在 `source/images/icons/`

3. **草稿管理**
   - 使用 `make draft` 创建草稿
   - 草稿完成后用 `make publish` 发布

### 项目维护

1. **定期更新 CHANGELOG.md**

3. **使用 VS Code 推荐扩展**
